# MM Finance Consulting

Стартовый проект сайта финансовой консалтинговой компании MM Finance Consulting.

## Stack

- Vite + React + TypeScript
- Tailwind CSS через `@tailwindcss/vite`
- `react-i18next` и `i18next` для RU/KZ локализации
- `react-router-dom`
- `lucide-react`
- `framer-motion`

## Scripts

```bash
npm run dev
npm run build
npm run lint
```

## Deployment

Проект готов к деплою на Vercel как стандартное Vite-приложение.

- Build command: `npm run build`
- Output directory: `dist`
- Base path: `/`
